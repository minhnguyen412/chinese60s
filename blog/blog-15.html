[Real-Life Chinese] 如何使用洗衣机 - How to use a washing machine - Douyin Script

Laundry is something we all have to do, but explaining how to wash clothes in another language can be surprisingly challenging. In today’s lesson, we’ll go through a script where a young man explains step by step how to use a washing machine.

Let’s explore the video script together 👇

📽️ The Script: 如何使用洗衣机 - How to use a washing machine

Script with Pinyin and Translation

如何使用洗衣机
rú hé shǐ yòng xǐ yī jī
How to use a washing machine

首先把这些衣服均匀的放在这个洗衣机里面
shǒu xiān bǎ zhè xiē yī fú jūn yún de fàng zài zhè ge xǐ yī jī lǐ miàn
First, put these clothes evenly into the washing machine

然后就可以开始倒入洗衣液或洗衣粉。
rán hòu jiù kě yǐ kāi shǐ dào rù xǐ yī yè huò xǐ yī fěn
Then you can start pouring in laundry detergent or washing powder.

洗衣液倒一点就可以了。
xǐ yī yè dào yī diǎn jiù kě yǐ le.
Just pour in a little detergent.

我这一瓶我一般可以用个半年。
wǒ zhè yī píng wǒ yì bān kě yǐ yòng gè bàn nián.
This one bottle usually lasts me about half a year.

然后后面这个放水的...
rán hòu hòu miàn zhè ge fàng shuǐ de...
Then, the water inlet at the back…

这个水龙头记得是打开的。
zhè ge shuǐ lóng tóu jì de shì dǎ kāi de.
Remember to open this faucet.

然后一定要把盖子盖上。
rán hòu yí dìng yào bǎ gài zi gài shàng
And you must put the lid on.

然后这里有标准的启动。
rán hòu zhè lǐ yǒu biāo zhǔn de qǐ dòng.
Here there is the standard “start” button.

一般是摁住电源。
yī bān shì èn zhù diàn yuán.
Usually, you hold the power button.

然后就直接看这个水位
rán hòu jiù zhí jiē kàn zhè ge shuǐ wèi
Then just look directly at the water level

我的衣服差不多就刚好这个水位。
wǒ de yī fú chà bù duō jiù gāng hǎo zhè ge shuǐ wèi.
My clothes roughly match this water level.

然后直接摁住启动。
rán hòu zhí jiē èn zhù qǐ dòng.
Then press start directly.

它这里就会开始自动加水了。
tā zhè lǐ jiù huì kāi shǐ zì dòng jiā shuǐ le.
It will start adding water automatically.

开水加满之后要记得把盖子盖上。
rú guǒ róng yì xǐ huài de yào ná yí gè xǐ yī dài zhuāng qǐ lái.
If it’s easy to damage, put it in a laundry bag.

如果容易洗坏的要拿一个洗衣袋装起来。
rán hòu xǐ yī jī cái huì zhuàn dòng.
Then the washing machine will spin.

如果会掉色或者深色的衣服一般要分开洗。
rú guǒ huì diào sè huò zhě shēn sè de yī fú yì bān yào fēn kāi xǐ.
If clothes may fade or are dark-colored, usually wash them separately.

如果不会掉色的话就不用管可以直接放进去。
rú guǒ bú huì diào sè de huà jiù bú yòng guǎn kě yǐ zhí jiē fàng jìn qù.
If they won’t fade, you can just put them in directly.

如果在中途发现还有没放进的衣服盖子可以直接打开。
rú guǒ zài zhōng tú fā xiàn hái yǒu méi fàng jìn de yī fú gài zi kě yǐ zhí jiē dǎ kāi.
If you notice during the cycle that some clothes weren’t put in, you can open the lid.

它就会停了。
tā jiù huì tíng le.
It will stop.

然后你再把衣服放进去。
rán hòu nǐ zài bǎ yī fú fàng jìn qù.
Then you put the clothes inside.

然后再把盖子盖上。
rán hòu zài bǎ gài zi gài shàng.
Then put the lid back on.

它又会启动了。
tā yòu huì qǐ dòng le.
And it will start again.

📚 Grammar Structures to Notice

1. 把 (bǎ)

used to bring the object before the verb, emphasizing handling or change.
Meaning: “let/make you eat until satisfied.”
📝 Example: 把衣服放进去 → “Put the clothes inside.”

2. 差不多 (chà bù duō)

roughly, almost.
📝 Example: 我的衣服差不多刚好这个水位 → “My clothes roughly match this water level.”

3. 刚好 (gāng hǎo)

just right
📝 Example: 刚好这个水位 → “just at this water level.”

4. 如果 (rú guǒ)

if/condition marker.
📝 Example: 如果会掉色，就要分开洗 → “If clothes fade, wash separately.”

5. 装起来 (zhuāng qǐ lái)

put inside (and keep it contained).
📝 Example: 要拿一个洗衣袋装起来 → “Put it in a laundry bag.”

6. 放进去 (fàng jìn qù)

to put inside.
📝 Example: 直接放进去 → “Put it directly inside.”

🎯 Takeaway

You learned essential laundry vocabulary like 洗衣机 (washing machine), 洗衣液 (detergent), 水龙头 (faucet), and 洗衣袋 (laundry bag).

You practiced conditional grammar with 如果 (“if”) and resultative handling with 把.

Daily life Chinese often uses direct instructions (摁住, 放进去, 盖上) which are clear and action-oriented.

The script shows how to describe step-by-step processes in Chinese, very useful for tutorials or giving instructions.

Understanding these structures not only improves comprehension but also helps you explain your own daily routines in Chinese.

👉 Try repeating the sentences aloud, practice writing the characters, and even act out the scene with friends. With practice, you won’t just learn words — you’ll learn how to live the language.
